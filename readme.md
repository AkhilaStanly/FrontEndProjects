Dependencies needed:

Please run the following from root directory
npm init
npm install express
npm install body-parser
npm install express-session
npm install mongodb
npm install mongoose
npm install passport
npm install passport-local
npm install bcrypt
npm install nodemon

To run the server:
From root dir,
nodemon main.js
